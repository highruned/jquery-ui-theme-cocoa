# jQuery UI theme: "Cocoa"

Cocoa-style theme for jQuery UI.

## Getting started

* Clone the repo: `git clone git@github.com:ericmuyser/jquery-ui-theme-cocoa.git`
* Include the files from `build` in a theme directory as you normally would.
